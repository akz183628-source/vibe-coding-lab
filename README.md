# Vibe Coding Lab

Вайб кодинг тәжірибелері — AI арқылы жасалған жобалар, промпттар жəне эксперименттер.

## Папка структурасы

- `projects/` — клиентке немесе тəжірибе үшін жасалған сайттар
- `prompts/` — жақсы жұмыс істеген AI промпттар
- `snippets/` — қайта қолданатын код бөліктері
- `experiments/` — жаңа нəрселерді сынап көру
